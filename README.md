calc4
=====
### About this

指定した4つの数字を四則演算で10にするプログラム。

### How to use

コマンドライン変数に4つの数字を渡してください。   
10を作れる場合はその演算方法を表示します。   

### supplementary

ちゃんと作ることに挫折して文字列からDataTable.Computeメソッドを使うという裏技的手法を用いています。
